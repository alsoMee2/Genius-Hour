A for loop is a loop
Heres how it works:
for(var x = 0; x < 10; x++){
//double slashes are comments and JavaScript will ignore everything after it
// any command to be looped
}
x can be replaced with anything you want
var x = 0 is declaring a variable x with a value of 0
x < 10 means repeat this loop as long as x is less than 10
x++ is shorthand for x = x + 1 which just means add 1 to x;
