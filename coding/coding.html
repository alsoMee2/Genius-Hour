how to colour something
fill(__,__,__)
what it does: it colours everything after it.
quick tip: you can just put fill(0); for black and fill(255); for white.
note:the 1st is red, the 2nd is green, the 3rd is blue. 0 is none of that color, 255 is all of that color. Putting a number more than 255 is pointless because if the number is more than 255 it just counts it as 255.
Common mistakes: forgetting the ,s, forgetting the ; at the end, typos.

Common mistakes: forgetting ,s, forgetting ; at the end, typos.
